# Django Accounts App

A reusable Django app providing user authentication: registration, login, logout, and session management.  
Designed as part of the **quotes project** or as a standalone module for any Django project.

## 🚀 Features

- User registration
- User login
- User logout
- Login required decorators for protected views
- Django session management
- Clean structure ready for extension with profiles or email confirmation

## 🏗️ Technologies

- Python 3.8+
- Django 4.x
- SQLite (default Django DB)
